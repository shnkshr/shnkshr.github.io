Detailed Description

This class represents an abstraction over the internal engine renderer.

This class initializes the ResourceManager class with the correct renderer, which in this case is SDL_Renderer. Currently just provides a reference to the main Game window and renderer to simplify code in GameRunner class. Basically this class deals with all SDL (the internal graphics engine used) related functionality.
